# UQ-PIGAN
Part of code and data of our paper

If you want to run this code, please create a folder and name it "train1", and put the "train_data1.csv" into this folder. 

Additionally, please install the required package in our code. Anaconda and Pycharm are recommended.

Then, run the "UQ_PIGAN.py" to train our model.

You can visualize the generated data by running "evaluate.py".
